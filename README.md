## Getting Started

### Prerequisites

Project dependencies 

* Netwide Assembler
* GNU C Compiler
* GNU Linker
* QEMU - System i386
* GNU Make
* Genisoimage
* GDB

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Installation


1. Clone the repo
   ```sh
   git clone hhttps://github.com/labsister21/os-2024-gyattbelajar
   ```
2. Build and run the program
   ```sh
   make
   ```
